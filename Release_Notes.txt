1.  Introduction

   This document identifies changes between Embedded
   Hypervisor releases and a list of limitations.

-------------------------------------------------------
2.  Compatibility/Dependencies

   -embedded-hv v0.5-003 has been tested with Simics 4.2.14 / pre14
    and the P4080DS board

   -in sync with Embedded Hypervisor Architecture specifciation
    0.878

-------------------------------------------------------
3.  Detailed List of Changes

   -Topaz 0.5-003 bug fixes and changes
      -ELF loaders: clean up and better error handling and
       diagnostics
      -bug 4233: start/restart combination caused partition to
       hang
      -bug 4202: copy_between_gphys: flush caches.  This will
       provide proper cache synchronization for partman
       based image loading.

   -Topaz 0.5-002 bug fixes and changes
      -bug 4091: gdb is not flushing caches on breakpoint writes
      -bug 3470: partman load returns zero for success
      -bug 3739: check to verify that MSI subwindow lies within
       PCI-E inbound ATMU window
      -bug 4097: gtlb shell command improvements
      -bug 4126: don't use the CPCs if they are turned off
      -misc updates to byte-channel mux and queue infrastructure
       to fix some race conditions
      -added system 'reset' command to hypervisor shell
      -optimization to remove redundant PID invalidates of the TLB
      -misc unit test cleanup

   -Topaz 0.5-001 bug fixes and changes (from 0.4-011)
      -mux_server now supports an -exec command that
       allows connection to another program (such
       as skermit)
      -ns16550 initialization
      -workaround for CPU errata PAMU3
      -bug fix to optimized memset()
      -unit test updates:
         -support byte-channels as a console
         -initialize hardware UART
         -bcmux: Fix mismatched error code
