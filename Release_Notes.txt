1.  Introduction

   This document identifies changes between Embedded
   Hypervisor releases and a list of limitations.

-------------------------------------------------------
2.  Compatibility/Dependencies

   -embedded-hv v0.6-002 has been tested with Simics 4.2.36 / pre43
    and the P4080DS board

   -in sync with Embedded Hypervisor Architecture specifciation
    0.881

-------------------------------------------------------
3.  Detailed List of Changes

   -v0.6-002 bug fixes and changes
      -initial nap hcall support
      -initial error interrupt handling support
      -support partial writes for byte-channel send API (bug 5164)
      -implement virtual device driver framework
      -add additional event types to the hypervisor statistics
      -tlbivax deadlock bugfix (bug 5073)
      -PCI-E outbound ATMU fixups (enables 32-bit OS with PCI-E
       on 36-bit hypervisor)

   -v0.6-001 bug fixes and changes (since 0.5-008)
      -initial hypervisor statistics support in command shell
      -bug 4284-- improve robustness of getting crash dumps
       out of the hypervisor when the console is on a mux
      -added mechanism to get machine check info out to
       console
      -machine check handler improvements
      -support for shell command to dump guest physical memory
      -add FH_IDLE hcall
      -unit test cleanup to sync with sim.git
